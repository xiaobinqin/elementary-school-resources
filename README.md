# 小学生学习资源 📚

> 基于人教版教材，为三年级和四年级小学生精心整理的学习资源

## 项目简介

本项目整理了人教版三年级和四年级的**数学、语文、英语**三科学习资料，包含：

- 📖 每章节知识点总结与难点解析
- ✏️ 针对性练习题（每章6-10道）
- 🚀 拔高题目（每章4-5道）
- ✅ 详细答案与解析（点击"查看答案"按钮显示）

## 使用方法

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/xiaobinqin/elementary-school-resources.git
   ```
2. 用浏览器打开 `main.html` 文件
3. 点击对应年级 → 学科 → 册次 → 章节，即可查看学习内容
4. 每道练习题下方有"查看答案"按钮，点击可展开详细解析

## 特色

- 🎨 活泼可爱的界面设计，适合小朋友使用
- 📱 响应式布局，支持电脑和平板
- 🔍 知识点 + 难点 + 练习 + 拔高，体系完整
- 💡 答案详解采用按需显示，避免学生提前看到答案

## 生成方式

所有HTML文件由 `generate_part1.py` 脚本自动生成，共生成约110个HTML文件。
